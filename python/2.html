Variables y Tipos de Datos

¿Qué es una variable?

Una variable es un espacio en la memoria donde puedes guardar información. En Python puedes declarar variables solo poniendo su nombre y su valor

Ejemplo de variable

nombre = "Juan"
edad = 25
ciudad = "Madrid"

Tipos de datos básicos

  • String (cadenas de texto): "Hola mundo"
  • Integer (integer): 42
  • Float (flotante): 3.1415
  • Boolean (booleanos): true, false
  • Null: una variable sin valor

Mostrar variables en la consola

Puedes usar print() para mostrar datos en la consola:


mensaje = "¡Hola!"
print(mensaje)

¡Hola!
