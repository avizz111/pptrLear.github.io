Funciones

¿Qué es una función?

Una función es un bloque de código reutilizable que se puede ejecutar cuando se necesite. Nos ayuda a evitar la repetición y organizar el código.

Declarar una función

def nombre_funcion():
    # bloque de código
    return resultado_opcional

Llamar a una función

nombre_funcion(); // Esto ejecutara el bloque de código

Funciones con parámetros

def nombre_funcion(parámetros):
    # bloque de código
    return resultado_opcional

Funciones que retornan un valor

def sumar(a, b):
    return a + b

resultado = sumar(3, 5)
print(resultado)  # 8

Ejemplo


def saludar(nombre="amigo"):
    print(f"Hola, {nombre}!")

            

Hola, amigo!
