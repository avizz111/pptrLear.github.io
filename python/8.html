Curso de Python - Lección 8

Objetos

¿Qué es un objeto?

Un objeto es una estructura que permite agrupar datos y funcionalidades relacionadas mediante pares clave-valor.

Declaración de una clase

Una clase es donde se guardan todos los atributos del objeto

class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

Declaración de un objeto

Teniendo ya preparada las clases necesarias se puede crear el objeto con las clases que ocupe

persona1 = Persona("Ismael", 30)

Acceder a propiedades

print(persona1.nombre)  # Ismael
print(persona1.edad)    # 30

Modificar propiedades

persona1.edad = 29;

Agregar nuevas propiedades

persona1.pais = "México";

Agregar metodos

Los metodos son las acciones que los objetos pueden hacer

def saludar(self):
    return f"Hola, soy {self.nombre}"
