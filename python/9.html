Condicionales

¿Qué son los condicionales?

Los condicionales son estructuras que permiten ejecutar diferentes bloques de código dependiendo de si se cumple o no una condición.

Sentencia if

if condición:
    # bloque de código si la condición es verdadera

Sentencia if-else

if edad >= 18:
    print("Acceso permitido")
else:
    print("Acceso denegado")

Sentencia elif

nota = 8

if nota >= 9:
    print("Excelente")
elif nota >= 7:
    print("Notable")
elif nota >= 6:
    print("Suficiente")
else:
    print("Insuficiente")
