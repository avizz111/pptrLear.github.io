Random

¿Qué es el Random?

El Random es una biblioteca ampliamente utilizada para aprender a usar bibliotecas y crear eventos de aleatoridad

Las biblioteca son fragmentos de codigo ya escritas para poder ser reutilizadas en multiples ocasiones

Tipos de estado

Existen distintos tipos de estado en esta biblioteca los cuales son:


random.random() #Número decimal entre el 0 y el 1 sin incluir estos
random.randint(a, b) #Entero entre a y b incluyendo estos
random.randrange(start, stop, step) #Entero aleatorio como range
random.choice(lista) #Elige un elemento aleatorio de una secuencia
random..shuffle(lista) #Mezcla los elementos de una lista in situ
random.sample(lista, k) #Devuelve una nueva lista con k elementos aleatorios sin repetir
random.uniform(a, b) #Número decimal aleatorio entre a y b

Llamar a Random

Antes de cualquier cosa primero debes llamar todas las bibliotecas a utilizar y luego escribi tu código

Para llamar la biblioteca random se inicia de la siguiente forma

import random
