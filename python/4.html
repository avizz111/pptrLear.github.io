Condicionales

¿Qué son las condicionales?

Las estructuras condicionales son otros tipos de operadores que permiten ejecutar diferentes bloques de código dependiendo de si una condición es verdadera o falsa.

Operadores de comparación

  • == Igualdad
  • != Desigualdad
  • === Igualdad estricta
  • >, <, >=, <= Comparaciones numéricas

Operadores Lógicos

  • and (y)
  • or (o)
  • not (no)

Operador ternario

Una forma corta de escribir una condición:

valor_si_verdadero if condición else valor_si_falso

Ejemplo


edad = 27
mensaje = "Niño" if edad >= 0 and edad <=18 else "Adulto"
print("Eres" + mensaje)
            

Eres Adulto
