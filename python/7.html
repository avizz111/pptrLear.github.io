Arreglos

¿Qué es un arreglo?

Un arreglo (array) es una estructura que permite almacenar múltiples valores en una sola variable. Puedes acceder a los elementos por su posición (índice).

Declaración de un arreglo

 frutas = ["manzana", "banana", "naranja"];

Acceder a elementos

print(frutas[0]); // "manzana"

Modificar elementos

frutas[1] = "pera";

Agregar y eliminar elementos

frutas.push("kiwi");    // Agrega al final
frutas.pop();           // Elimina el último
frutas.add("uva");  // Agrega al inicio
frutas.pop(0);         // Elimina el primero

Ejemplo


dulces = ["paleta", "caramelo", "chocolate"]
dulces.pop()
dulces.push("chicle")
print(dulces)
            

["paleta", "caramelo", "chicle"]
