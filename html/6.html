Curso 6: Enlaces en HTML

Aprende a conectar páginas web, archivos y sitios externos usando enlaces.

¿Qué es un enlace?

Un enlace o hipervínculo es un elemento que permite a los usuarios navegar de una página web a otra o abrir documentos, correos y más.

La etiqueta <a>

La etiqueta <a> (anchor) se usa para crear enlaces.

<a href="https://www.ejemplo.com">Visita nuestro sitio</a>

Visita nuestro sitio

Atributo href

  • href: define la dirección a donde lleva el enlace.
  • target: define dónde se abre (por ejemplo, _blank abre en una nueva pestaña).
  • title: muestra un texto cuando se pasa el cursor sobre el enlace.

Tipos de enlaces

  • Externos: Llevan a otra página web.
    • <a href="https://openai.com">OpenAI</a>
  • Internos: Llevan a otra parte de la misma web o a otro archivo HTML local.
    • <a href="contacto.html">Contacto</a>
  • Correo: Abre el cliente de correo.
    • <a href="mailto:alguien@email.com">Enviar correo</a>
  • Teléfono: Abre la app para hacer una llamada (en móviles).
    • <a href="tel:+521234567890">Llámanos</a>

Enlace con imagen

También puedes convertir una imagen en un enlace:

<a href="https://google.com">
  <img src="https://via.placeholder.com/150" alt="Google">
</a>
