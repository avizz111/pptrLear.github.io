Curso 3: Enlaces y Navegación

Aprende a conectar páginas web y navegar por la web

¿Qué es un enlace?

Los enlaces (o hipervínculos) permiten a los usuarios navegar de una página a otra.

Se crean usando la etiqueta <a>.

Enlaces a otras páginas

Para enlazar a otra web:

<a href="https://www.google.com">Ir a Google</a>

El atributo href contiene la URL del destino.

Enlaces internos

Sirven para navegar dentro del mismo sitio web:

<a href="contacto.html">Ir a la página de contacto</a>

Se usa cuando tienes varias páginas en tu sitio.

Abrir enlaces en nueva pestaña

Se usa el atributo target="_blank" para abrir un enlace en otra pestaña:

<a href="https://www.openai.com" target="_blank">Visitar OpenAI</a>

Enlaces dentro de la misma página

También puedes saltar a secciones dentro de la misma página:

<a href="#final">Ir al final de la página</a>

...

<h2 id="final">Este es el final</h2>
