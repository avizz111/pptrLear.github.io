Curso 2: Texto en HTML

Aprende a mostrar contenido textual en la web

Encabezados

HTML ofrece 6 niveles de encabezado que van del más importante <h1> al menos importante <h6>: 

<h1>Título principal</h1>
<h2>Subtítulo</h2>
<h3>Encabezado nivel 3</h3>
...
<h6>Encabezado pequeño</h6>

Párrafos

Los párrafos se crean con la etiqueta <p>:

<p>Este es un párrafo de ejemplo.</p>

También puedes usar <br> para hacer saltos de línea:

Primera línea.<br>
Segunda línea.

Formato de texto

Estas etiquetas te permiten modificar el estilo del texto:

  • <b>: Negrita (visual)
  • <strong>: Negrita semántica
  • <i>: Cursiva (visual)
  • <em>: Cursiva semántica
  • <u>: Subrayado
  • <mark>: Texto resaltado

Ejemplo combinado:

<p>Esto es <strong>muy importante</strong> y <em>debe notarse</em>.</p>

Listas

Hay dos tipos principales de listas:

Lista no ordenada:
<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>
Lista ordenada:
<ol>
  <li>Primero</li>
  <li>Segundo</li>
</ol>
