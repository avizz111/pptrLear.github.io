Curso 1: Introducción al HTML

Aprende la base de todas las páginas web

¿Qué es HTML?

HTML significa HyperText Markup Language. Es el lenguaje principal que se usa para estructurar contenido en la web. Define encabezados, párrafos, imágenes, enlaces, formularios y más.

Estructura básica de un documento HTML

Todo documento HTML inicia con una estructura como esta:

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
  </body>
</html>

Etiquetas esenciales

  • <!DOCTYPE html>: Informa al navegador el tipo de documento.
  • <html>: Contenedor de todo el contenido HTML.
  • <head>: Contiene información como título, codificación, estilos.
  • <body>: Contiene lo que se mostrará en la página.

Comentarios en HTML

Los comentarios sirven para dejar notas dentro del código y no se muestran al usuario:

<!-- Esto es un comentario -->

¿Cómo se visualiza una página HTML?

Puedes crear un archivo con extensión .html y abrirlo en cualquier navegador moderno (Chrome, Firefox, Edge, etc.). Por ejemplo, guarda este contenido como index.html y ábrelo: 

<!DOCTYPE html>
<html>
  <head><title>Ejemplo</title></head>
  <body>
    <h1>Hola desde mi primera página web</h1>
  </body>
</html>
