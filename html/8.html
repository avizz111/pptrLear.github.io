Curso 8: Formularios en HTML

Aprende a crear formularios para recolectar datos del usuario.

¿Qué es un formulario?

Un formulario permite a los usuarios introducir datos que se pueden enviar a un servidor para procesarlos.

Etiqueta <form>

Se utiliza para definir un formulario. Dentro de ella van los campos de entrada.

<form action="procesar.php" method="post">
  <input type="text" name="nombre">
  <input type="submit" value="Enviar">
</form>

Campos comunes

  • <input type="text"> - Campo de texto
  • <input type="email"> - Correo electrónico
  • <input type="password"> - Contraseña
  • <textarea> - Área de texto grande
  • <select> y <option> - Menú desplegable
  • <input type="radio"> - Opción única
  • <input type="checkbox"> - Selección múltiple

Ejemplo de formulario

Atributos importantes

  • required: Hace que el campo sea obligatorio
  • placeholder: Texto de ayuda en el campo
  • name: Nombre del campo (para el servidor)
  • value: Valor predefinido
  • method y action: Controlan el envío
