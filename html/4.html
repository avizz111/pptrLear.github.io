Curso 4: Imágenes en HTML

Aprende cómo insertar y configurar imágenes en tus páginas web.

¿Por qué usar imágenes?

Las imágenes son una parte fundamental del diseño web. Ayudan a comunicar ideas visualmente, mejorar la estética del sitio y captar la atención del usuario.

Etiqueta <img>

La etiqueta <img> se usa para insertar imágenes. No tiene etiqueta de cierre.

<img src="ruta-de-la-imagen.jpg" alt="Descripción de la imagen">
  • src: especifica la ruta o URL de la imagen.
  • alt: texto alternativo que se muestra si la imagen no se carga (y útil para accesibilidad).

Atributos comunes

  • width y height: definen el tamaño de la imagen.
  • title: muestra un texto al pasar el cursor sobre la imagen.

Ejemplo

<img src="https://via.placeholder.com/300" alt="Imagen de ejemplo" width="300">

Imágenes desde internet

Puedes usar imágenes desde otros sitios web, pero asegúrate de tener permiso para usarlas o que sean de uso libre.

Imágenes locales

También puedes usar imágenes guardadas en tu proyecto. Por ejemplo:

<img src="imagenes/logo.png" alt="Logo">
