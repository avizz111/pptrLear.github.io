Condicionales

¿Qué son las condicionales?

Las estructuras condicionales son otros tipos de operadores que permiten ejecutar diferentes bloques de código dependiendo de si una condición es verdadera o falsa.

Operadores de comparación

  • == Igualdad
  • != Desigualdad
  • === Igualdad estricta
  • >, <, >=, <= Comparaciones numéricas

Operadores Lógicos

  • && AND (y)
  • || OR (o)
  • ! NOT (no)

Operador ternario

Una forma corta de escribir una condición:

condición ? valor_si_verdadero : valor_si_falso;

Ejemplo


let edad = 20;
let mensaje = edad >= 18 ? "Adulto" : "Menor";
console.log("Eres " + mensaje); // Eres Adulto
