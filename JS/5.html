Bucles

¿Qué son los bucles?

Los bucles permiten repetir una serie de instrucciones varias veces. Existen diferentes tipos de bucles en JavaScript.

Bucle for

for (let i = 0; i < 5; i++) {
  console.log("Repetición " + i);
}

Bucle while

let i = 0;
while (i < 5) {
  console.log("Iteración " + i);
  i++;
}

Bucle do...while

let i = 0;
do {
  console.log("Valor: " + i);
  i++;
} while (i < 5);
