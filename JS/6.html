Funciones

¿Qué es una función?

Una función es un bloque de código reutilizable que se puede ejecutar cuando se necesite. Nos ayuda a evitar la repetición y organizar el código.

Declarar una función

function saludar() {
  console.log("¡Hola, mundo!");
}

Llamar a una función

saludar(); // Esto imprimirá "¡Hola, mundo!"

Funciones con parámetros

function saludarUsuario(nombre) {
  console.log("Hola, " + nombre + "!");
}
saludarUsuario("Carlos");

Funciones que retornan un valor

function sumar(a, b) {
  return a + b;
}
let resultado = sumar(3, 5); // resultado = 8

Llama una funcion desde tu documento

Puedes llamar las distintas funciones que hagas desde tu archivo con el atributo onclick = "", esto lo que hace es que al presionar dicho elemento llamara a la funcion que este entre las comillas

<button onclick = "funcion_ejemplo()">
function funcion_ejemplo() {

    alert("hola");
}
