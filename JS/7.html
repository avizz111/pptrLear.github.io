Arreglos

¿Qué es un arreglo?

Un arreglo (array) es una estructura que permite almacenar múltiples valores en una sola variable. Puedes acceder a los elementos por su posición (índice).

Declaración de un arreglo

let frutas = ["manzana", "banana", "naranja"];

Acceder a elementos

console.log(frutas[0]); // "manzana"

Modificar elementos

frutas[1] = "pera";

Agregar y eliminar elementos

frutas.push("kiwi");    // Agrega al final
frutas.pop();           // Elimina el último
frutas.unshift("uva");  // Agrega al inicio
frutas.shift();         // Elimina el primero
