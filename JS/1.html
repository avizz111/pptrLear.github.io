Introducción a JavaScript

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite crear funcionalidades dinámicas en sitios web, como botones que reaccionan al hacer clic, formularios interactivos, animaciones, y más.

¿Dónde se usa?

  • Validación de formularios
  • Animaciones en sitios web
  • Manipulación del DOM
  • Aplicaciones web completas (con frameworks como React, Vue o Angular)

¿Cómo se incluye en HTML?

Puedes incluir JavaScript directamente dentro de tu documento HTML o mediante archivos externos.

<script>
  alert('¡Hola desde JavaScript!');
</script>
