Variables y Tipos de Datos

¿Qué es una variable?

Una variable es un espacio en la memoria donde puedes guardar información. En JavaScript, puedes declarar variables usando let, const o var.

Ejemplo de variable

let nombre = "Juan";
const edad = 25;
var ciudad = "Madrid";

Tipos de datos básicos

  • String (cadenas de texto): "Hola mundo"
  • Number (números): 42, 3.14
  • Boolean (booleanos): true, false
  • Undefined: una variable sin valor
  • Null: una variable intencionalmente vacía

Mostrar variables en la consola

Puedes usar console.log() para mostrar datos en la consola del navegador:

let mensaje = "¡Hola!";
console.log(mensaje);
