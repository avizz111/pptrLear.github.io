Condicionales

¿Qué son los condicionales?

Los condicionales son estructuras que permiten ejecutar diferentes bloques de código dependiendo de si se cumple o no una condición.

Sentencia if

if (condicion) {
  // Código a ejecutar si la condición es verdadera
}

Sentencia if-else

if (condicion) {
  // Código a ejecutar si la condición es verdadera
} else {
  // Código a ejecutar si la condición es falsa
}

Sentencia else-if

if (condicion1) {
  // Código si condicion1 es verdadera
} else if (condicion2) {
  // Código si condicion2 es verdadera
} else {
  // Código si ninguna de las condiciones es verdadera
}
