Curso de JavaScript - Lección 10

DOM

¿Qué es el DOM?

El DOM es un termino que se usa para poder acceder a las distintas propiedades de un objeto

Obtener objeto por ID

Para poder acceder a los objetos de tus paginas debes de hacerlo mediante su ID

document.getElementById("Ejemplo");

Guardar los objetos de tus documentos dentro de una variable

Puedes guardar tus objetos en variables para poder acceder a ellos varias veces

et variable = document.getElementById("Ejemplo");

Accede a las propiedades

Una vez guardados en variables los objetos puedes acceder a sus propiedades para visualizarlas

let variable = document.getElementById("Ejemplo");

console.log(variable.value);

Cuando ya puedes acceder a las propiedades puedes cambiarlas para que tus paginas realmente sea dinámicas

let variable = document.getElementById("Ejemplo");


variable.value = "Nuevo texto";

Ejemplo interactivo

Ingresa un texto que que quieras que aparezca

Aquí aparecera el nuevo texto

function mostrarMensaje() {
    const texto = document.getElementById("numero").value;
    let mensaje = texto;
    document.getElementById("textoIngresar").textContent = mensaje;
}
