1.¿Qué es un algoritmo?

Un programa informático completo. Un lenguaje de programación específico. Una secuencia de pasos para resolver un problema.

2.¿Qué hace un compilador?

Ejecuta directamente el código fuente. Interpreta el código línea por línea. Traduce el código fuente a código máquina

3. ¿Qué es una variable?

Una constante que no cambia su valor. tipo de dato específico. espacio en memoria que almacena un valor.

4.¿Qué es un bucle (loop)?

estructura de control que ejecuta un bloque de código una sola vez. Un tipo de dato complejo. Una estructura de control que ejecuta un bloque de código repetidamente.

5.¿Qué es una función (o procedimiento)?

Un tipo de dato primitivo. Un elemento de la interfaz gráfica de usuario. Un bloque de código que realiza una tarea específica.

6.¿Qué es la recursividad?

Un método de ordenamiento de datos. Una función que se llama a sí misma

7. ¿Qué es un array (o vector)?

Una estructura de datos que almacena elementos de diferentes tipos. Una estructura de datos que almacena una colección de elementos del mismo tipo.

8. ¿Qué es un puntero?

Un tipo de dato de cadena. Una variable que almacena la dirección de memoria de otra variable.

9.¿Qué es la depuración (debugging)?

El proceso de compilar el código. El proceso de encontrar y corregir errores en el código.

10.¿Qué es un comentario en el código?

Un tipo de error de sintaxis. Una línea de texto que explica el código, pero no afecta su ejecución.

11.¿Qué significa "código fuente"?

El código ejecutable del programa. El código en lenguaje máquina. El código escrito por el programador en un lenguaje de programación.

12.¿Qué es un condicional (if-else)?

Un tipo de bucle. Un tipo de dato. Una estructura de control que ejecuta un bloque de código basado en una condición.

13.¿Qué es una estructura de datos?

Un lenguaje de programación. Un tipo de variable Una forma de organizar y almacenar datos.

14.¿Qué es la herencia (en programación orientada a objetos)?

un tipo de dato un tipo de bucle Un mecanismo que permite a una clase heredar propiedades y métodos de otra clase.

15.¿Qué es un objeto (en programación orientada a objetos)?

Una variable simple. Una función. Una instancia de una clase.

16.¿Qué es una clase (en programación orientada a objetos)?

Una tipo de bucle Una tipo de complemento Una plano o molde para crear objetos

17. ¿Qué es el polimorfismo (en programación orientada a objetos)?

Un tipo de error. Un tipo de dato La capacidad de un objeto de tomar diferente forma.

18.¿Qué es el encapsulamiento (en programación orientada a objetos)?

Un tipo de bucle. Un tipo de variable. La capacidad de ocultar los datos internos de una clase.

19. ¿Qué es un algoritmo de búsqueda?

Un tipo de dato. Un método para ordenar datos. Un método para encontrar un elemento específico en una colección de datos.

20.¿Qué es un algoritmo de ordenamiento?

Un método para encontrar un elemento específico. Un tipo de estructura de datos. Un método para organizar datos en un orden específico (ej. ascendente o descendente).
