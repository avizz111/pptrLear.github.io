Curso de CSS - Lección 2

Selectores y propiedades básicas

¿Qué son los selectores?

Los selectores permiten aplicar estilos a elementos HTML específicos. Determinan a qué elementos se les aplican las reglas CSS.

Tipos de selectores más comunes:

  • Selector de etiqueta: Aplica estilo a todos los elementos de un tipo. Ej: p { color: blue; }
  • Selector de clase: Aplica estilo a los elementos con una clase específica. Ej: .mi-clase { font-size: 18px; }
  • Selector de ID: Aplica estilo a un elemento con un ID único. Ej: #mi-id { text-align: center; }

¿Qué son las propiedades?

Las propiedades definen qué aspecto cambiarás: color, tamaño de fuente, márgenes, etc.

Ejemplos prácticos:

/* Selector de etiqueta */
h1 {
  color: darkgreen;
  font-size: 32px;
}

/* Selector de clase */
.destacado {
  background-color: yellow;
  padding: 10px;
}

/* Selector de ID */
#principal {
  text-align: center;
  font-weight: bold;
}

Aplicación en HTML:

<h1>Título</h1>
<p class="destacado">Este texto está destacado</p>
<div id="principal">Contenido principal</div>

Consejos:

  • Usa clases para aplicar estilos a varios elementos.
  • Evita abusar de los IDs para estilo, resérvalos para casos únicos.
  • Combina selectores para obtener más control (ej: div p).
