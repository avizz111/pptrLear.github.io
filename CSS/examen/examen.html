1. ¿Qué significa CSS?

Cascading Style Sheets (Hojas de Estilo en Cascada). Significa "Creative Styling Software". Significa "Cascading Server Scripts".

2. ¿Cuál es la función principal de CSS?

Su función principal es procesar formularios enviados por los usuarios. Su función principal es gestionar la lógica de base de datos de un sitio web. Separar la presentación (aspecto visual) del contenido (HTML) de una página web.

3. ¿Cómo se vincula un archivo CSS a una página HTML?

Se vincula copiando y pegando el código CSS directamente en la etiqueta <body> de HTML. Se vincula usando la etiqueta <script> con el atributo src. <link rel="stylesheet" href="estilo.css"> dentro de la etiqueta <head>

4. ¿Qué son los selectores CSS?

Son comandos de JavaScript que permiten manipular el DOM. Son etiquetas HTML especiales para definir el contenido del pie de página. Patrones que identifican elementos HTML a los que aplicar estilos.

5. tres tipos de selectores CSS

Selector de función, selector de variable, selector de bucle. Selector de imagen, selector de sonido, selector de video. Elemento ( p ,  div ), clase ( .miClase ), ID ( #miID ).

6.¿Cuál es la diferencia entre una clase y un ID en CSS?

Una clase se usa para elementos de texto, y un ID para elementos multimedia. Los ID son para estilos de solo lectura, mientras que las clases permiten estilos dinámicos. ID es único por página; clase puede usarse en múltiples elementos.

7.¿Cómo se define un estilo en línea en HTML?

Se define creando un archivo .inline-css y vinculándolo. Se define usando la etiqueta <style> dentro de la etiqueta <body>. Atributo  style  dentro de la etiqueta HTML (ej:  <p style="color:red;"> ).

8.¿Qué es la propiedad display en CSS?

Es una propiedad para definir la velocidad de carga de las imágenes. Es una propiedad para ajustar el brillo de la pantalla del usuario. Controla cómo se muestra un elemento (ej:  block ,  inline ,  none ).

9.¿Qué es la propiedad float en CSS?

Es una propiedad para hacer que los elementos floten sobre el agua en animaciones 3D. Es una propiedad para determinar si un elemento debe estar fijo o desplazarse con la página. Flota un elemento a izquierda o derecha (se recomienda usar  flexbox  o  grid ).

10.¿Qué es position: relative en CSS?

Significa que el elemento desaparecerá si no tiene contenido. Significa que el elemento siempre se posicionará en relación con la parte superior izquierda de la pantalla. Posiciona un elemento relativamente a su posición normal.

11.¿Qué es position: absolute en CSS?

Significa que el elemento no puede ser movido de ninguna manera. Significa que el elemento se ajustará automáticamente al tamaño del navegador. Posiciona un elemento relativamente a su padre con  position  no  static .

12.¿Qué es position: fixed en CSS?

Significa que el elemento está roto y no se puede mover. Significa que el elemento solo se muestra en dispositivos móviles. Posiciona un elemento fijo en la ventana del navegador.

13.¿Qué es la propiedad width en CSS?

Define la cantidad de bordes que tiene un elemento. Define el peso del elemento en megabytes. Define el ancho de un elemento.

14.¿Qué es la propiedad height en CSS?

Define el grosor del texto dentro del elemento. Define el número de hijos que puede tener un elemento. Define la altura de un elemento.

15.¿Qué es la propiedad margin en CSS?

Es el espacio interno de un elemento que se rellena con color. Es el espacio de seguridad para los elementos con contenido peligroso. Espacio exterior alrededor de un elemento.

16.¿Qué es la propiedad padding en CSS?

Es el relleno que se usa en los elementos con contenido de imagen. Es el espacio entre un elemento y el siguiente elemento en la página. Espacio interior dentro de los bordes de un elemento.

17.¿Qué es una unidad de medida común en CSS?

Kilómetros por hora (km/h). Gigabytes (GB). Píxel (px).

18.¿Qué es un comentario en CSS?

Es una etiqueta HTML para ocultar elementos de la vista del usuario. Es un código que el navegador ejecuta para optimizar la carga.  /* Comentario */ .

19.¿Qué es la cascada en CSS?

Es un error común en CSS que hace que los estilos se apliquen de forma aleatoria. Es una animación de flujo de agua en la página web. Determina qué reglas de estilo se aplican cuando hay conflictos. Las más específicas tienen prioridad.

20.¿Dónde puedo encontrar más información sobre CSS?

En enciclopedias de jardinería o libros de cocina. En foros de discusión sobre cómo reparar televisores viejos. MDN Web Docs, W3Schools, Codecademy.
