Colores, fondos y bordes

Uso de colores en CSS

CSS permite aplicar colores a texto, fondos, bordes y más. Puedes usar:

  • Colores por nombre: red, blue, green
  • Hexadecimales: #ff0000, #00ff00
  • RGB: rgb(255, 0, 0)
  • RGBA (con opacidad): rgba(0, 0, 255, 0.5)

Ejemplo:

h1 {
  color: #1a73e8;
}

p {
  background-color: rgba(255, 255, 0, 0.3);
}

Fondos en CSS

Con CSS puedes aplicar colores, imágenes o gradientes como fondo.

Ejemplos:

body {
  background-color: #f0f0f0;
  background-image: url('fondo.jpg');
  background-size: cover;
  background-repeat: no-repeat;
}

También puedes usar gradientes:

div {
  background: linear-gradient(to right, red, yellow);
}

Estilo de bordes

Los bordes en CSS pueden ser sólidos, punteados, dobles, etc.

Ejemplo:

.caja {
  border: 2px dashed #198754;
  padding: 10px;
  margin: 10px;
}

Propiedades relacionadas:

  • border-width, border-style, border-color
  • border-radius para bordes redondeados
