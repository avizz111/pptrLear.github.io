Curso de CSS - Lección 4

Fuentes y texto

Fuentes en CSS

CSS permite personalizar las fuentes utilizadas en tu sitio web para mejorar su legibilidad y apariencia.

Propiedades de fuente más comunes:

  • font-family: Tipo de fuente
  • font-size: Tamaño de letra
  • font-weight: Grosor del texto (ej: normal, bold)
  • font-style: Estilo del texto (normal, italic)
body {
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-weight: normal;
  font-style: italic;
}

Texto en CSS

También puedes modificar cómo se ve el texto con propiedades como:

  • text-align: alineación (left, center, right)
  • text-decoration: subrayado, tachado
  • text-transform: mayúsculas, minúsculas
  • line-height: espacio entre líneas
h1 {
  text-align: center;
  text-transform: uppercase;
}

p {
  text-decoration: underline;
  line-height: 1.5;
}

Importar fuentes externas

Puedes usar Google Fonts u otras fuentes externas:

@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');

body {
  font-family: 'Roboto', sans-serif;
}
