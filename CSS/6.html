Modelo de Caja (Box Model)

¿Qué es el Modelo de Caja?

En CSS, todos los elementos HTML se consideran cajas rectangulares. El modelo de caja describe el diseño y el espacio de estos elementos.

Partes del modelo de caja:

  • Content: el contenido del elemento (texto, imagen, etc.)
  • Padding: espacio entre el contenido y el borde
  • Border: el borde del elemento
  • Margin: espacio fuera del borde, separación con otros elementos

Ejemplo de CSS:

.caja {
  width: 200px;
  padding: 20px;
  border: 5px solid #198754;
  margin: 30px;
}

Visualización del modelo de caja:

Propiedad box-sizing

Por defecto, el tamaño de los elementos no incluye padding ni border. Puedes cambiar eso con:

* {
  box-sizing: border-box;
}

Esto hace que el ancho y alto incluyan padding y border, facilitando el diseño.
