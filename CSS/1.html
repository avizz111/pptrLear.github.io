Curso de CSS - Lección 1

Introducción a CSS

¿Qué es CSS?

CSS (Cascading Style Sheets, u Hojas de Estilo en Cascada) es un lenguaje que permite aplicar estilos a documentos HTML. Gracias a CSS, podemos definir cómo se verá el contenido de una página web.

¿Para qué sirve?

  • Aplicar colores, fuentes, márgenes, bordes, y más.
  • Crear diseños adaptables y profesionales.
  • Separar el contenido del diseño.
  • Reutilizar estilos en varias páginas con un solo archivo CSS.

Formas de utilizar CSS

  1. CSS en línea: dentro de la etiqueta HTML con el atributo style.
  2. CSS interno: en el documento HTML dentro de una etiqueta <style>.
  3. CSS externo: enlazando un archivo .css mediante la etiqueta <link>.

Ejemplo de CSS interno:

<style>
  body {
    background-color: lightblue;
  }

  h1 {
    color: navy;
    text-align: center;
  }
</style>

Sintaxis de CSS

CSS se basa en selectores y declaraciones. Por ejemplo:

selector {
  propiedad: valor;
}

Ejemplo real:

p {
  font-size: 16px;
  color: black;
}
