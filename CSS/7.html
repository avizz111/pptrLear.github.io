Curso de CSS - Lección 7

Bordes, márgenes y rellenos

Bordes en CSS

Los bordes definen los límites visuales de los elementos. Puedes controlar su tamaño, estilo y color.

border: 2px solid #198754;

También puedes personalizar cada lado:


border-top: 2px dashed red;
border-right: 3px dotted blue;
border-bottom: 1px solid black;
border-left: 4px double green;

Márgenes (margin)

El margen es el espacio externo que separa un elemento de los demás.

margin: 10px;

Para márgenes individuales:


margin-top: 20px;
margin-right: 10px;
margin-bottom: 15px;
margin-left: 5px;

Relleno (padding)

El relleno es el espacio interno entre el borde y el contenido del elemento.

padding: 15px;

Para valores individuales:


padding-top: 10px;
padding-right: 20px;
padding-bottom: 10px;
padding-left: 20px;

Ejemplo completo:

.caja {
  border: 2px solid #198754;
  margin: 20px;
  padding: 15px;
}
