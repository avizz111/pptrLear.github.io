Diseño de Layouts

Diseño con Flexbox

Flexbox es una herramienta de CSS para crear diseños de página responsivos y flexibles.

Las propiedades clave de Flexbox son:

  • display: flex; para activar Flexbox en un contenedor.
  • justify-content: Alineación de los elementos a lo largo del eje principal.
  • align-items: Alineación de los elementos a lo largo del eje transversal.
  • flex-wrap: Permite que los elementos se envuelvan en varias líneas si es necesario.

Ejemplo básico de Flexbox:

.contenedor {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.elemento {
  flex: 1;
  margin: 10px;
}

Diseño con Grid

CSS Grid Layout es una herramienta poderosa para crear layouts en dos dimensiones.

Las propiedades clave de Grid son:

  • display: grid; para activar Grid en un contenedor.
  • grid-template-columns: Define el número de columnas y su tamaño.
  • grid-gap: Define el espacio entre las filas y las columnas.

Ejemplo básico de Grid:

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.grid-item {
  background-color: #198754;
  height: 100px;
}
