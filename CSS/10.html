Curso de CSS - Lección 10

Transiciones y Animaciones

¿Qué son las transiciones?

Las transiciones permiten que los cambios de propiedades CSS ocurran de forma gradual.

Propiedades comunes:

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay

Ejemplo de transición:

.caja {
  background-color: #198754;
  transition: all 0.5s ease;
}

.caja:hover {
  background-color: #20c997;
  transform: scale(1.2);
}

¿Qué son las animaciones?

Las animaciones permiten crear movimientos personalizados con la regla @keyframes.

Ejemplo de animación:

@keyframes mover {
  0%   { transform: translateX(0); }
  50%  { transform: translateX(100px); }
  100% { transform: translateX(0); }
}

.elemento {
  animation: mover 2s infinite;
}
