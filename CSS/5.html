Unidades de medida y tamaños

Unidades de medida en CSS

CSS permite usar diferentes tipos de unidades para definir tamaños. Se dividen en dos grupos principales:

Unidades absolutas:

  • px: píxeles
  • cm, mm: centímetros, milímetros
  • in: pulgadas
  • pt, pc: puntos, picas

Unidades relativas:

  • %: porcentaje relativo al elemento padre
  • em: relativo al tamaño de fuente del elemento padre
  • rem: relativo al tamaño de fuente del html
  • vw, vh: ancho y alto del viewport

Ejemplo:

div {
  width: 80%;
  font-size: 1.2em;
  padding: 2rem;
  margin: 5vh;
}

Tamaño de elementos

Para controlar el tamaño de los elementos puedes usar:

  • width: ancho
  • height: alto
  • max-width, min-width: tamaño máximo y mínimo
  • max-height, min-height: altura máxima y mínima
.caja {
  width: 300px;
  height: 200px;
  max-width: 100%;
}
