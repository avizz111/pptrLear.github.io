Colores y Fondos

Colores en CSS

Para aplicar colores en CSS, podemos usar varios formatos:

  • Nombre del color: red, green, blue, etc.
  • Hexadecimal: #FF5733 (Rojo anaranjado)
  • RGB: rgb(255, 87, 51)
  • RGBA: rgba(255, 87, 51, 0.5) (Con transparencia)
  • HSL: hsl(9, 100%, 60%) (Tono, Saturación, Luminosidad)

Ejemplo de uso de colores:

body {
  background-color: #f0f0f0;
}

h1 {
  color: rgb(255, 87, 51);
}

button {
  background-color: rgba(0, 123, 255, 0.7);
}

Fondos en CSS

Los fondos en CSS se pueden definir de varias maneras:

  • Color de fondo: background-color
  • Imagen de fondo: background-image
  • Repetir imagen: background-repeat
  • Ajustar imagen: background-size
  • Posición de fondo: background-position

Ejemplo de fondo con imagen:

body {
  background-color: #f0f0f0;
  background-image: url('https://example.com/imagen.jpg');
  background-repeat: no-repeat;
  background-size: cover;
  background-position: center;
}
