Curso de SQL - Lección 1

Que es SQL y que es SELECT

¿Qué es SQL?

SQL (Structured Query Language) es un lenguaje de programación diseñado para gestionar y manipular datos en bases de datos relacionales. SQL se utiliza para crear, modificar y consultar bases de datos, y es un estándar ampliamente adoptado en la industria.

¿para que sirve?

  • 1.Crear y modificar estructuras de datos: Crear tablas, índices, vistas, etc.
  • 2.Insertar, actualizar y eliminar datos: Agregar, modificar o eliminar registros en una base de datos.
  • 3.Consultar datos: Recuperar información específica de una base de datos utilizando consultas.
  • 4.Realizar operaciones de agregación: Calcular sumas, promedios, máximos, mínimos, etc.

¿Que es SELECT?

SELECT es una instrucción SQL que se utiliza para consultar datos en una base de datos. Permite seleccionar columnas específicas de una o varias tablas y devuelve los registros que coinciden con los criterios de búsqueda.

SELECT columna1, columna2, ...
FROM tabla;

Donde: - columna1, columna2, etc. son las columnas que se desean seleccionar. - tabla es la tabla de la que se desean seleccionar los datos.
