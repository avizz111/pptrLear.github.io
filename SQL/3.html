UPDATE

¿Qué es un UPDATE?

UPDATE es una instrucción SQL que se utiliza para modificar registros existentes en una tabla de una base de datos.

Sintaxis basica

La sintaxis básica de UPDATE es la siguiente:

UPDATE tabla
SET columna1 = valor1, columna2 = valor2, ...
WHERE condición;

Donde:

  • tabla es la tabla en la que se desean modificar los registros.
  • columna1, columna2, etc. son las columnas que se desean actualizar.
  • valor1, valor2, etc. son los nuevos valores que se desean asignar a las columnas correspondientes.
  • condición es la condición que se utiliza para seleccionar los registros que se desean actualizar.
