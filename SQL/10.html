Subconsultas

¿Qué son las subconsultas?

Una subconsulta es una consulta SQL que se utiliza dentro de otra consulta. Las subconsultas permiten realizar operaciones más complejas y flexibles en las bases de datos.

Tipos de subconsultas

Existen varios tipos de subconsultas, como:

  • Subconsultas escalares: devuelvenun solo valor
  • Subconsultas de fila: devuelven solo una fila
  • Subconsultas de tabla: devuelven una tabla de resultados

Uso de subconsultas

Las subconsultas se pueden utilizar en diferentes partes de una consulta, como:

  • WHERE
  • FROM
  • SELECT

Ejemplo

SELECT * FROM clientes
WHERE edad > (SELECT AVG(edad) FROM clientes);
