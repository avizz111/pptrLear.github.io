Curso de SQL - Lección 2

INSERT INTO

¿Qué es un INSERT INTO?

INSERT INTO es una instrucción SQL que se utiliza para agregar nuevos registros a una tabla en una base de datos.

Sintaxis basica

La sintaxis básica de INSERT INTO es la siguiente:


INSERT INTO tabla (columna1, columna2, ...)
VALUES (valor1, valor2, ...);

Donde:

- tabla es la tabla en la que se desean agregar los registros.

- columna1, columna2, etc. son las columnas de la tabla que se desean llenar con datos.

- valor1, valor2, etc. son los valores que se desean insertar en las columnas correspondientes.

ejempo

1. Insertar un nuevo registro en una tabla: 

INSERT INTO clientes (nombre, email, edad)
VALUES ('Juan', 'juan@example.com', 30);
Esto agrega un nuevo registro a la tabla clientes con los valores especificados. 2. Insertar varios registros a la vez: 
INSERT INTO clientes (nombre, email, edad)
VALUES
('Juan', 'juan@example.com', 30),
('María', 'maria@example.com', 25),
('Pedro', 'pedro@example.com', 40);
Esto agrega varios registros a la tabla clientes con los valores especificados.
