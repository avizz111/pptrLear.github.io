INDEX

¿Que es el INDEX?

INDEX es una estructura de datos que se utiliza en bases de datos para mejorar la velocidad de las consultas que buscan registros específicos en una tabla.

¿Qué es un índice?

Un índice es una copia de una o varias columnas de una tabla, ordenada de manera que permita una búsqueda rápida y eficiente de los registros que coinciden con una condición específica.

Tipos de índices

  • 1.Índice único: Garantiza que no haya valores duplicados en la columna indexada.
  • 2.Índice no único: Permite valores duplicados en la columna indexada.
  • 3. Índice compuesto: Se crea sobre varias columnas de una tabla.

Cuándo crear un índice

  • 1.Columnas que se utilizan frecuentemente en consultas: Si una columna se utiliza frecuentemente en consultas, puede ser beneficioso crear un índice sobre ella.
  • 2.Columnas que tienen una alta selectividad: Si una columna tiene una alta selectividad (es decir, muchos valores únicos), puede ser beneficioso crear un índice sobre ella.
