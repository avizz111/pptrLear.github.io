ORDER BY

¿Qué es ORDER BY?

Es una clausula SQLque se utiliza para ordenar los registros de una tablaen una base de datos según una o varias columnas

Sintaxis básica

SELECT * FROM tabla
ORDER BY columna [ASC:DESC]

Donde

  • tabla es la tabla de la que se desean seleccionar registros
  • columna es la columna según la cual se desea ordenar los registros
  • ASC es el orden predeterminado que ordena los registros de menor a mayor
  • DESC ordena los registros de mayor a menor
