Deleted

¿Qué es el DELETE?

DELETE es una instrucción SQL que se utiliza para eliminar registros de una tabla de una base de datos

Sintaxis básica

La sintaxis básica de DELETE es la siguienta:

DELETE FROM tabla
WHERE condición

Donde:

  • tabla es la tabla de la que se desea eliminar registros
  • condición es la condición que se utiliza para seleccionar los registros que se desean eliminar

Precauciones

Es importante que pongas el WHERE para evitar todos los registros, como recomendacion siempre hacer respaldos antes de ejecutar esta sentencia ya que este cambio es irreversible
