GROUP BY

¿Qué es el GROUP BY

GROUP BY es una cláusula SQL que se utiliza para agrupar registros de una tabla en función de una o más columnas.

Sintaxis básica

La sintaxis básica de GROUP BY es la siguiente:

SELECT columna1, columna2, ...
FROM tabla
GROUP BY columna1, columna2, ...;

donde:

tabla es la tabla de la que se desean agrupar registros.>

columna1, columna2, etc. son las columnas que se utilizan para agrupar los registros.

Ejemplo

Agrupar registros por país y calcular la suma de ventas:

SELECT pais, SUM(ventas) AS total_ventas
FROM clientes
GROUP BY pais;
